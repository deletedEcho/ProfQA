Команды для навигации и базовой работы с файлами

  • ls — показывает содержимое текущей директории. Используйте ls с адресом папки, чтобы показать содержимое конкретной директории. Добавьте -R для вывода файлов из поддиректорий.
  • cd — сменить директорию. Используйте cd с адресом, чтобы перемещаться между папками. Сокращенные варианты: cd- (перемещение в предыдущую директорию) или cd (прыжок в домашнюю папку).
  • pwd — показать полный путь текущей директории.
  • file — определить тип файла и получить информацию о нем, включая данные EXIF для изображений.
  • mv — перемещение файлов. Пример: mv timeweb.txt ~/Desktop (перенос на рабочий стол) или mv oldname.txt newname.txt (переименование файла).
  • rm — удаление файлов и директорий. Внимание: данные уничтожаются без возможности восстановления. Пример: rm ~/Timeweb (удаление директории Timeweb).
  • rmdir — создание пустой директории. Пример: mkdir ~/Desktop/New_Project.
  • mkdir — удаление пустой директории. rmdir не трогает другие файлы и субдиректории.
  • cat — вывод содержимого текстовых файлов. Пример: cat filename.txt.
